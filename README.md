# YT Kids Timer

A child-friendly YouTube Kids player with parental session controls,
built with Qt6/C++ for Linux (Zorin OS, Fedora, Ubuntu).

## Features
- Session timer with grace period logic
- Parental controls (PIN-protected, scheduled access)
- Child-selectable artwork wallpaper
- Warning dialogs at 5 and 1 minute remaining

## Build
```bash
cmake -B build && cmake --build build
```
